Прицеп для перевозки гидроцикла (водного мотоцикла)

Длиной до 3,9 м****, массой до 460 кг.

Одноосный, без тормозной системы.

Рессорная подвеска.

Исполнение

  • Колеса 13" или 16"

Стандартная комплектация

  • Ложементы обитые ковролином - 2 шт. длиной по 1,7 м - служат опорой для гидроцикла, предотвращают повреждение корпуса перевозимой водной техники. Положение ложементов регулируется под обводы гидроцикла;
  • Таль ручная (лебёдка) облегчает погрузку гидроцикла;
  • Носовой упор перемещается вдоль дышла, регулируется по высоте;
  • Держатель вилки.

Дополнительное оборудование

  • Опорная ножка с резиновым колесом, регулируемая по высоте;
  • Проставка дышла – изменение высоты сцепного устройства;
  • Закатной ролик - облегчит погрузку гидроцикла;
  • Трап-мото (трапы для перевозки двухколёсного мотоцикла);
  • Кузов для перевозки грузов (внутренние размеры, мм: 1570х1110х272);
  • Кронштейн запасного колеса, запасное колесо 13", 16";
  • Чехол для лебедки;
  • Подножки.

Технические характеристики

  • Антикоррозийное цинковое покрытие, нанесенное методом горячего цинкования, обеспечит защиту на долгие годы;
  • Рессорная подвеска с гидравлическими амортизаторами обеспечивает плавность хода прицепа во всем диапазоне;
  • Предусмотрены петли для крепления техники.
